# BUSE AUTO AI HR Recruitment System

An AI-powered HR recruitment system designed for BUSE to automate resume parsing and candidate screening, generating an optimized shortlist for HR teams to interview.

---

## 🚀 Key Features

*   **Automated Resume Parsing:** Extracts key skills, education, and experience from uploaded candidate CVs automatically.
*   **Intelligent Screening & Filtering:** Utilizes matching algorithms to evaluate candidate profiles against specific job descriptions.
*   **Optimized Shortlisting:** Generates a ranked list of top-tier candidates, eliminating manual sorting for HR personnel.
*   **Human-in-the-Loop Workflow:** Smoothly hands off the finalized shortlist to HR teams for final interview scheduling and selection.

*   **Frontend:** [e.g., React.js / HTML & Tailwind CSS ]
*   **Backend:** [e.g., Node.js]
*   **Database:** [e.g., MySQL]
*   **AI/Parsing Engine:** [e.g. OpenAI API] 

---

## 📋 System Architecture & Workflow

1.  **Upload:** Candidates or HR upload resumes (PDF/Word format).
2.  **Process:** The system parses the text and extracts core professional data points.
3.  **Evaluate:** The algorithm matches candidate data against the specified hiring criteria.
4.  **Shortlist:** HR receives a clean, filtered dashboard displaying the best-matched applicants.
